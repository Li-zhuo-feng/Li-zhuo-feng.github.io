python最简讲义

1.基础语法

  • 注释用 # 表示开始
  • 用缩进（TAB）表示代码块从属关系
  • 单个脚本文件中缩进必须统一

2.变量与赋值

  • python不用指定变量的类型（int，double，char）
  • 用 = 来进行赋值
  • 变量名字由字母，数值和下划线（ _ ）组成，但不能由数字开头
  • 变量名对大写敏感
  • python变量可以用中文字符命名（但不建议这么做）
  • 函数名、类名和变量名都叫做标识符，python标识符都要符合以上命名规则

3.数据结构

3.1字典

创建字典：d = {key1 : value1, key2 : value2, key3 : value3 }

调用字典：d[key1]+d[key2]

修改字典：d[key1] = new_value

删除字典元素： del d[key1]

清空字典：dict.clear()

删除字典： del d

3.2元组

  • 元组可以看做是不用自己取名字的字典
  • 元组的调用序号用 0 到 n-1表示（n为元组内元素的个数）
  • 元组也可以倒着调用，用 -1 到 -n 表示
  • 元组不能修改
  • 元组可以用运算符进行运算（+、-、*、/）

3.3列表

  • 列表可以看作是可以进行修改的元组

3.4字符串

  • 字符串可以看作元素连起来的列表
  • ‘ ’ ，” “，‘’‘ ’‘’，”“” “”“ 都可以用来表示字符串，但后两个可以换行
  • python不会运行一个常量也不会报错，所以可以用字符串来进行注释（特别是多行注释）

3.5切片

  • 元组，列表，字符串都可以进行切片索引

4.控制结构语句

4.1条件语句

基本语法结构：

  • elif 和 else 都可以省略不写
  • 判断条件的结果应该为布尔值——True或False
  • 0判断为False，不为零的数字判断为True
